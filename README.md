## Descripcion
Este repositorio contiene el codigo del trabajo final que permite diseñar e implementar un sistema orientado a objetos completo utilizando:
- Abstracción / Encapsulamiento / Ocultamiento
- Herencia / Polimorfismo
- Relaciones entre objetos
- Colecciones
- Persistencia mediante ORM
- Base de datos MariaDB/mySQL
